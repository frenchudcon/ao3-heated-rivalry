# Assets Heated Rivalry – AO3 Skin

Ce dossier contient les **assets visuels** (PNG) à héberger sur GitHub Pages pour le skin AO3 "Heated Rivalry".

## ⚠️ Important : droits & usage

- **Pas de logos officiels** (équipes, ligues, marques).
- **Pas de Snoopy officiel** → utilise une silhouette générique de beagle.
- Tous les assets sont **inspirés du lore** du fandom, pas des visuels officiels.

---

## 📐 Formats & export

- **Icônes** : 512×512 px, PNG transparent
- **Textures** : 1920×1080 px, PNG transparent ou semi-transparent
- **Export** : PNG-24 avec transparence

---

## 🚀 Hébergement GitHub Pages

Une fois les assets créés :

1. Place-les dans ce dossier `assets/`.
2. Active GitHub Pages sur ton repo (Settings → Pages).
3. Les assets seront accessibles via :
   ```
   https://TON-USERNAME.github.io/ao3-heated-rivalry/assets/nom-du-fichier.png
   ```
4. Remplace `TON-GH-PAGES` dans les CSS par cette URL de base.

## 📝 Notes

- Les assets sont **optionnels** : le skin fonctionne sans images (tout est en CSS pur).
- Si tu n'as pas les assets, le skin utilisera uniquement les effets CSS (grain SVG inline, gradients, etc.).
- Les assets ajoutent une couche visuelle supplémentaire mais ne sont pas obligatoires pour le fonctionnement de base.
